# Olist Global E-Commerce Performance Engine

📌 **Interactive Dashboard:** [👉 Click Here to Interact with Live Tableau Dashboard](https://public.tableau.com/views/Olist_Ecommerce_Optimization_Engine/Dashboard1?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

## 💼 The Business Problem
Retail operations needed a centralized analytics engine to identify shipping fulfillment bottlenecks across regional states, track fluctuating monthly revenue streams, and measure repeat customer purchase velocity to target platform retention initiatives.

## 🛠️ Tech Stack
- **Database Engine:** PostgreSQL (Data Storage & Advanced Relational Querying)
- **Business Intelligence Platform:** Tableau Public (Spatial Mapping & KPI Dashboarding)

## 💡 Key Business Insights Discovered
- **Logistics Bottlenecks:** Remote territories like Amazonas (AM) and Acre (AC) suffer from the highest average transit delays past delivery estimates, signaling a critical need to distribute inventory to localized regional fulfillment hubs.
- **Customer Velocity:** Repeat customer retention maintains a baseline return interval of 77.9 days, capturing 3,345 transactions across the current lifecycle.
### 📊 Dataset Source
The raw data utilized for this project is sourced from the [Kaggle Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce).
